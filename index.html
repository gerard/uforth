uforth by gerard

A Forth interpreter in ARM assembly

uforth is a small project meant for my spare time. My goal is to have some, more or less, maintanable ASM code base and while I’m on it, I try to attach to DPANS94, altough violations don’t bother me right now, other than missing functionality.

Dependencies

No dependencies are needed to run the binary on an arm machine. Not even a C library. The only supported platform is Linux. For building you are going to need an assembler. My toolchain ishere. 

  deb http://www.emdebian.org/debian lenny main
To run it on PC, you can use qemu. If you want to run the tests you need the following perl modules:
  • Config::IniFiles
  • Expect

Compilation

If you have the cross utils from emdebian (as and ld), this should work: 

$ make uforth
If you have a different toolchain, define $AS and $LD, like this: 
$ make uforth AS=/path/to/as LD=/path/to/ld
To run the tests: 
$ make check
To just run everything: 
$ make
